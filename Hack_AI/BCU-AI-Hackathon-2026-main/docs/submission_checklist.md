# Submission Checklist

Before submitting your GitHub link, check that your repository includes:

- [* ] `HACK-AI.csv`
- [ *] Source code and/or notebooks
- [ *] PowerPoint/PDF presentation
- [ *] README with run instructions
- [ *] Qwen2.5-7B via Ollama WITH TRANSFER LEARNING 
- [ ] No organiser answer key or private files
- [*] `HACK-AI.csv` has exactly 100 rows and columns `question_no,answer`
- [* ] All answers are one of `A`, `B`, `C`, `D`, or `E`

Suggested GitHub structure:

```text
your_team_repo/
  README.md
  TEAMNAME_submission.csv
  src/
  notebooks/
  slides/
```
